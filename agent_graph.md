# Agent Graph

User Query
   ↓
Planner
   ↓
Data Agent
   ↓
Insight Agent
   ↓
Evaluator
   ↓
Creative Generator
   ↓
Report Builder

Flow description:
- Planner decomposes tasks.
- Data Agent summarizes numeric signals.
- Insight Agent forms hypotheses.
- Evaluator verifies them quantitatively.
- Creative Generator produces improved messaging.
- Orchestrator writes outputs to reports/.
